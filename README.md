Smart Waste Management System
Smart Waste Management System is a GUI-based application developed using **C++** and the **Raylib graphics library**. The project is designed to simulate waste management operations in different city areas by allowing users to monitor waste categories, clean locations, and track cleaning status through an interactive graphical interface. The system manages different types of waste, including organic, plastic, metal, glass, and non-recyclable waste. Users can select an area, perform cleaning operations, view waste details, and check whether an area has been cleaned. The application utilizes Object-Oriented Programming (OOP) concepts such as classes, objects, encapsulation, and data structures to efficiently organize and manage information. This project demonstrates the practical application of C++ programming, GUI development, and waste management simulation while providing a simple and user-friendly experience.

Tools & Technologies Used
- C++
- Raylib Graphics Library
- Object-Oriented Programming (OOP)
- STL (Map Data Structure)
- Visual Studio / Visual Studio Code

Key Features
- Interactive GUI built with Raylib
- Area-based waste management simulation
- Waste categorization and monitoring
- Area cleaning functionality
- Cleaning status tracking
- Multiple predefined city locations
- User-friendly navigation system
- Real-time waste information display
